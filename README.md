## 开发笔记

### 本地开发

```bash
$ pnpm i
$ pnpm run dev
$ open http://localhost:7001/
```

### 部署

```bash
$ pnpm start
```

### 内置指令

- 使用 `pnpm run lint` 来做代码风格检查。
- 使用 `pnpm test` 来执行单元测试。
